# php_simple_routing

## How to start:

[1] open routes.php  
[2] add your desired routings  
[3] dont forget the .htacces file !  

###### Documentation

> **Main Page**  
> any('/', 'filepath');

> **POST**  
> post('/postPicture/$id', 'filepath');

> **GET**  
> get('/getPicture/$id', 'filepath');


> **with $id for example you can pass a dynamic variable in the url =)**
